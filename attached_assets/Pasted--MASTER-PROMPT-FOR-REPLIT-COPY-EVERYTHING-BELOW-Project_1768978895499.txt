Project Title

AI-Based Personalized Study Plan Generator using Syllabus Analysis and LLM-Driven Familiarity Assessment

Project Overview

Build a full-stack web application that automatically generates a personalized study timetable for a student based on their uploaded syllabus, initial familiarity with the subject, available study time, and exam deadline.

The system must not rely on manual admin intervention, must be scalable, and must use LLMs only for question generation, not for decision-making logic.

The project is intended for a B.Tech CSE major project and journal publication, so all logic must be explainable, rule-based, and realistic.

High-Level Workflow

Student registers/logs in

Student uploads syllabus (PDF or image)

System extracts text using OCR

NLP extracts subject, topics, and subtopics

System generates syllabus-specific MCQs using an LLM

Student takes a short familiarity test

System classifies learner as Slow / Medium / Fast

System analyzes topic difficulty

System generates a personalized, day-wise study plan

System adapts the plan based on student progress
